LeetCode 3：无重复字符的最长子串

用滑动窗口维护一个“当前不重复”的连续区间。右指针负责尝试扩张，遇到重复字符时左指针收缩窗口，直到窗口重新合法。

Java 滑动窗口 HashSet 双指针
O(n) 每个字符最多进窗口一次、出窗口一次

题目概述

给定一个字符串 s，请找出其中不含有重复字符的最长子串长度。

注意是子串，必须连续；不是子序列，不能跳着选字符。
输入 输出 解释
"abcabcbb" 3 最长为 "abc"
"bbbbb" 1 最长为 "b"
"pwwkew" 3 最长为 "wke"

考察知识点

滑动窗口

窗口表示当前正在检查的一段连续子串，始终尽量保持“不重复”。

HashSet 判重

Set<Character> 用来快速判断右指针字符是否已经在窗口里。

双指针边界

left 指向窗口左边界，right 指向下一个准备进入窗口的字符。

核心思路

看 s[right]
不重复：加入窗口
重复：移除 s[left]
更新最大长度

窗口含义可以记成：[left, right)，左闭右开。也就是说，right 还没加入窗口，它只是“下一个要尝试加入的字符”。

当 s[right] 已经在 Set 中时，不能直接跳过它，因为重复字符可能还在窗口中间。稳妥做法是不断移动 left，每次移除一个左侧字符，直到 s[right] 不再重复。

Java 参考实现

import java.util.HashSet;
import java.util.Set;

class Solution {
    public int lengthOfLongestSubstring(String s) {
        // 窗口定义为 [left, right)，也就是 left 已经在窗口内，
        // right 指向下一个准备尝试加入窗口的字符。
        int left = 0;
        int right = 0;

        // length 始终等于当前窗口长度；也可以用 right - left 计算。
        int length = 0;
        int maxLength = 0;

        // characters 保存当前窗口里已经出现过的字符。
        // 它必须和 [left, right) 这段窗口内容保持完全一致。
        Set<Character> characters = new HashSet<>();

        while (right < s.length()) {
            char current = s.charAt(right);

            if (!characters.contains(current)) {
                // current 不重复时，才可以加入窗口，并把 right 向右推进。
                // 推进后窗口扩大一位，所以 length 增加，并更新答案。
                characters.add(current);
                right++;
                length++;
                maxLength = Math.max(maxLength, length);
                continue;
            }

            // current 已经在窗口中，说明窗口内存在重复字符。
            // 此时不要移动 right；先从左侧逐个移除字符，直到 current 不再重复。
            characters.remove(s.charAt(left));
            left++;
            length--;
        }

        return maxLength;
    }
}
变量 含义 关键点
left 窗口左边界 重复时右移，并从 Set 中删除离开的字符
right 准备进入窗口的字符下标 只有成功加入字符时才右移
length 当前窗口长度 也可以用 right - left 替代
maxLength 历史最大窗口长度 只在窗口扩张成功后更新

为什么不是暴力枚举

暴力做法

枚举每个起点，再向右检查是否重复。最坏情况下会反复扫描同一批字符，时间复杂度可能到 O(n^2)。

滑动窗口

窗口只向右移动，不回头。每个字符最多被 right 加入一次，被 left 移除一次，所以整体是 O(n)。

易错点

  1. 把子串和子序列混淆：这题必须连续，窗口天然保证连续。
  2. 重复时直接移动 right：这样会让 Set 里仍然保留重复字符，窗口状态会乱。
  3. 忘记从 Set 删除左侧字符：left 移动时，窗口外的字符必须同步移除。
  4. maxLength 更新时机错误：只有成功加入新字符，窗口变长后，再更新最大值。
  5. 空字符串：s.length() == 0 时循环不进，直接返回 0。

复杂度分析

时间复杂度：O(n)

left 和 right 都只会从左向右移动，最多各移动 n 次。

空间复杂度：O(min(n, 字符集大小))

HashSet 保存当前窗口中的字符，最多保存不重复字符的数量。

一句话记忆

右指针负责扩张，左指针负责排重；窗口合法时更新答案。
